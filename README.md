# 🎭 meme-loan - 金融讽刺静态页

> 张张嘴，摇摇头，智商税秒到账

一个讽刺非正规借贷行为的互动式静态网页项目，通过夸张的"人脸识别"流程，揭露某些金融APP的荒诞审核机制。

---

## 📁 文件结构

```
meme-loan-3/
├── index.html          # 落地页（极速放款广告）
├── face-check.html     # 人脸识别模拟页（摄像头互动）
├── result.html         # 结局页（真相揭露/恶搞）
├── style.css           # 全局样式（金融诈骗风）
├── script.js           # 核心交互逻辑
└── assets/
    └── sounds/         # 音效目录（可选）
```

---

## 🎨 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- TailwindCSS (CDN)
- Web Animations API
- MediaDevices API (摄像头调用)

---

## 📖 页面流程

1. **落地页** (`index.html`) - 夸张的金融广告
2. **人脸识别** (`face-check.html`) - 荒诞的动作检测
3. **结局页** (`result.html`) - 真相揭露 or 恶搞结局
