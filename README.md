# 🎭 meme-loan - 金融讽刺静态页（明亮版）

> 张张嘴，摇摇头，智商税秒到账

一个讽刺非正规借贷行为的互动式静态网页项目，通过夸张的"人脸识别"流程，揭露某些金融APP的荒诞审核机制。

---


## 📖 页面流程

1. **落地页** (`index.html`)
   - 夸张金融广告："极速放款，3分钟到账"
   - 倒计时 + 剩余名额（营造紧迫感）
   - 弹幕漂浮："审核通过率98%"、"秒到账"
   - 大按钮跳转到人脸识别页

2. **人脸识别模拟页** (`face-check.html`)
   - 纯前端动画模拟（无需摄像头权限）
   - 表情变化：😊 → 😮 → 😵 → 😉 → 🤔 → 😎
   - 提示语：张嘴、摇头、眨眼等荒诞要求
   - 扫描线动画 + 进度条
   - 完成后自动跳转结局页

3. **结局页** (`result.html`)
   - 恶搞结局："2024年度网贷受害者预备资格"
   - 信用分滚动：-9999 分（动画效果）
   - 讽刺文案 + 表情包 🥲
   - 两个按钮：查看反讽视频 / 学习金融知识
