🎭 关于这个项目

meme-loan-3 - 一个反讽非正规借贷的趣味静态网页

💡 项目灵感

本项目灵感来源于网络梗"张张嘴摇摇头立马贷"，通过夸张的金融广告、荒诞的人脸识别流程和恶搞的审核结果，讽刺某些非正规借贷APP的营销手段和审核机制。

✨ 核心功能

📣

落地页

夸张广告文案 + 动态倒计时 + 弹幕漂浮 + 30,000,000额度展示

📸

人脸模拟

纯动画模拟 + 表情变化 + 扫描线 + 荒诞提示语（张嘴/摇头/眨眼）

🎯

结局页

-9999信用分动画 + 3个权威跳转（反诈视频/案例/APP）+ 一键分享

🛠️ 技术实现

HTML5 CSS3 JavaScript ES6+ TailwindCSS Web Audio API

纯前端实现，无需后端，无需数据库，零依赖，可直接部署到任何静态托管平台。

⚠️ 安全与免责

本项目纯属艺术创作：
• 不提供任何借贷服务
• 不收集任何人脸数据
• 不开通任何金融功能
• 不与任何金融机构合作
• 所有内容均为虚构和讽刺

请勿模仿真实借贷行为，如需贷款请选择正规银行或持牌金融机构。
